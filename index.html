Projects

Drumonitor

Drumonitor is a program for Arch Linux based systems that will read and display different statistics about a user's computer. This is mainly used in cases where you want to set up a module in a program like tint2 or polybar. It is written in Rust to be as performative as possible. As of now the program is able to display CPU, RAM, and network stats, and can change how fast statistics are read and displayed. To read more about this, click the link below!
